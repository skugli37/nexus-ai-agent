# Tool Forge

Autonomni sistem za kreiranje i integraciju novih alata.

## Kada koristiti

- Otkriven pattern ponavljajućih zadataka (> 3x)
- Dream cycle predloži novi alat
- Korisnik zatraži novu funkcionalnost
- Postoji gap u dostupnim alatima
- Optimizacija workflow-a

**NOT for:**
- Jednokratni zadaci
- Jednostavne operacije
- Zadaci koji već imaju alate

### Phase 3: Code Generation

Generiši kod:

```python
#!/usr/bin/env python3
"""
Auto-generated by Tool Forge
Purpose: Sum numeric values in CSV column
Created: 2024-01-15
"""

import pandas as pd
import json
import sys
from pathlib import Path

def csv_column_sum(file_path: str, column_name: str, output_format: str = "plain"):
    """
    Calculate sum of values in a CSV column.
    
    Args:
        file_path: Path to CSV file
        column_name: Name of column to sum
        output_format: Output format (plain, json, table)
    
    Returns:
        dict with sum and count
    """
    # Validate inputs
    if not Path(file_path).exists():
        raise FileNotFoundError(f"File not found: {file_path}")
    
    # Read and process
    df = pd.read_csv(file_path)
    
    if column_name not in df.columns:
        raise ValueError(f"Column '{column_name}' not found. Available: {list(df.columns)}")
    
    column = df[column_name]
    numeric_mask = pd.to_numeric(column, errors='coerce').notna()
    numeric_values = pd.to_numeric(column[numeric_mask])
    
    result = {
        "sum": float(numeric_values.sum()),
        "count": int(len(numeric_values)),
        "mean": float(numeric_values.mean())
    }
    
    # Format output
    if output_format == "json":
        return json.dumps(result, indent=2)
    elif output_format == "table":
        return f"Sum: {result['sum']}\nCount: {result['count']}\nMean: {result['mean']}"
    else:
        return str(result['sum'])

if __name__ == "__main__":
    if len(sys.argv) < 3:
        print("Usage: csv_column_sum.py <file_path> <column_name> [output_format]")
        sys.exit(1)
    
    result = csv_column_sum(
        file_path=sys.argv[1],
        column_name=sys.argv[2],
        output_format=sys.argv[3] if len(sys.argv) > 3 else "plain"
    )
    print(result)
```

### Phase 4: Testing

Automatsko testiranje:

```python
# tests/unit/test_csv_column_sum.py

import pytest
import tempfile
import os
from generated.scripts.csv_column_sum import csv_column_sum

class TestCSVColumnSum:
    
    @pytest.fixture
    def sample_csv(self):
        content = "name,value,extra\na,10,x\nb,20,y\nc,30,z"
        with tempfile.NamedTemporaryFile(mode='w', suffix='.csv', delete=False) as f:
            f.write(content)
            yield f.name
        os.unlink(f.name)
    
    def test_basic_sum(self, sample_csv):
        result = csv_column_sum(sample_csv, "value")
        assert result == "60.0"
    
    def test_json_output(self, sample_csv):
        result = csv_column_sum(sample_csv, "value", "json")
        parsed = json.loads(result)
        assert parsed["sum"] == 60.0
        assert parsed["count"] == 3
    
    def test_missing_column(self, sample_csv):
        with pytest.raises(ValueError):
            csv_column_sum(sample_csv, "nonexistent")
    
    def test_missing_file(self):
        with pytest.raises(FileNotFoundError):
            csv_column_sum("/nonexistent/file.csv", "value")
```

## Best Practices

1. **Minimum occurrences** - Kreiraj alat samo za > 3 ponavljanja
2. **Test coverage** - Minimum 80% test coverage
3. **User approval** - Zatraži odobrenje pre integracije
4. **Documentation** - Uvek generiši docs
5. **Version control** - Prati verzije alata
6. **Rollback** - Omogući povratak na prethodnu verziju

## Bezbednost

- Generisani kod se čuva u izolovanom direktorijumu
- Svi alati se testiraju pre integracije
- Code review zahtevan za kompleksne alate
